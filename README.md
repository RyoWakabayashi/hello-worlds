# hello-worlds

Hello world on multiple languages

## Requirements

- asdf
- Docker

## Set up

```shell
asdf plugin-add nodejs
asdf plugin-add python
asdf install
npm install
pip install --requirement requirements.txt
asdf reshim python
pre-commit install
```

## Lint

### Node.js

```shell
npm run lint
```

### Python

```shell
pylint python
```
